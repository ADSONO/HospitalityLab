Logbook

Project Journal

A chronological record of experiments, progress, failures and small victories.

2026-06-09

Backend and API foundations

Most of the day focused on software development and research. Early backend components and API structure started to take shape.

2026-06-08

Infrastructure and frontend progress

The wall opening was enlarged for additional cables. TV and satellite equipment started to be installed more cleanly in the garage. A large part of the afternoon was spent on frontend development.

2026-06-07

A normal Sunday

The dish was moved closer to its intended long-term location, coaxial cables were routed more cleanly, and work resumed on the public website.

2026-06-06

Satellite finder received

The satellite finder made Astra alignment much easier and helped validate the temporary dish setup. Code snippets and VLAN planning also started to be reorganized.

2026-06-05

First satellite reception

The Vu+ Solo² receiver arrived, F-type connectors were installed and the first satellite channels were received. Televes equipment still refused to fully cooperate.

2026-05-31

HospitalityLab begins

Repository created, initial structure set up and the decision made to document existing hardware, experiments, code and ideas properly.
