# HospitalityLab Website v2

Multi-page version of the HospitalityLab website.

## Pages

- `index.html`
- `infrastructure.html`
- `logbook.html`
- `contact.html`

## Images

Place your photos here:

```text
assets/images/
├── satellite-dish.jpg
├── tv-headend.jpg
├── power-cabinet.jpg
└── astra-signal.jpg
```

If an image is missing, the website will automatically show an icon placeholder instead.

## GitHub Pages

Upload the folder as `website/` in the repository, then enable GitHub Pages from:

```text
Settings → Pages → Deploy from branch → main → /website
```
