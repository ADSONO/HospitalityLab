# Documentation

General documentation, ideas, notes and project planning.

## About

HospitalityLab is a personal project exploring the technologies and infrastructure behind modern hospitality businesses.

The goal is to design, build and document the different systems commonly found in hotels and campings, from management software and networking to television systems and electrical infrastructure.

## Scope

- Hotels
- Camping
- Property Management Systems
- Booking Platforms
- Networking
- Television & Satellite Systems
- Electrical Infrastructure
- Automation
- And more...

## Philosophy

Because building a website and playing with a network switch wasn't enough.
