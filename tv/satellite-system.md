# Current Satellite System

## Overview

This document describes the current satellite television infrastructure being developed within HospitalityLab.

Unlike the terrestrial TV system, the satellite setup is still a work in progress and several components are currently under evaluation.

## Current Equipment

### Satellite Dish

100 cm satellite dish currently installed for testing purposes.

### LNB

The current installation uses a standard single-output LNB.

While sufficient for basic testing, it is not suitable for the intended multiswitch-based distribution architecture.

A quattro LNB has been ordered and will replace the current unit.

### Televes T.0X Series

Several Televes T.0X modules recovered from previous installations.

These units are currently being tested and evaluated.

Progress has been slower than expected due to configuration challenges and limited documentation.

### Televes NevoSwitch

Televes NevoSwitch multiswitch intended to serve as the core of the future satellite distribution system.

Testing is currently limited by the existing LNB configuration.

### Satellite Finder

A dedicated satellite finder has been ordered to simplify dish alignment and troubleshooting.

## Current Status

🚧 Work in Progress

The satellite infrastructure is not yet operational.

Current efforts are focused on:

- Understanding the Televes T.0X platform
- Replacing the existing LNB with a quattro model
- Improving satellite alignment
- Building a proper multiswitch-based distribution architecture

## Notes

At the time of writing, the Televes T.0X equipment remains the most challenging part of the installation.

The hardware is clearly capable, but it is not always eager to cooperate.
